数组相关操作阅读[MDN文档](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/Array)

### 创建数组
```javascript
// 字面量
var arr = []
// 构造函数
var arr1 = new Array(1,2,3,4,5)
var arr2 = new Array(5); // 长度为5，元素全部为空
```

### 读写数组
```javascript
// 读写都使用 [] 
var arr = [1,2,3,4]
arr[1]; // 2
arr[4] = 5;
```

### 字符串生成数组
```javascript
var str = 'test';
var arr = str.split('')
```
